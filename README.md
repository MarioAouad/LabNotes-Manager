# LabNotes Manager

A streamlined desktop GUI application for managing and archiving laboratory notes. This tool simplifies the process of recording experiment data, ensuring consistent formatting for dates and subjects, and provides robust options for data export and database management.

## üöÄ Key Features

* **Structured Note Taking:** Log date, subject, and detailed notes in a centralized interface.
* **Smart Formatting:**
    * **Subjects:** Automatically capitalizes words and replaces spaces with underscores (e.g., "chemistry experiment" -> "Chemistry_Experiment").
    * **Dates:** Enforces strict DD-MM-YYYY format validation.
* **Data Management:**
    * **Show All:** View a list of all recorded subjects and dates.
    * **Details View:** Select any entry to read the full notes in a dedicated window.
* **SQL Export:** Seamlessly export your entire database to a clean `.sql` file (`lab_notes_export.sql`) for backups or migration.
* **Database Reset:** Built-in "Reset" function to clear all data and start fresh, with safety confirmation prompts.

## üéÆ How to Use

1.  **Add Note:** Enter the Subject, Date (DD-MM-YYYY), and your Notes. Click **Add Note** to save.
2.  **View Notes:** Click **Show All Notes** to see a list of entries. Select one and click **Show Notes** to read the details.
3.  **Export:** Click **Export to SQL** to save a backup of your data to `lab_notes_export.sql`.
4.  **Reset:** Use **Reset Database** to wipe all data (Warning: this cannot be undone).
